CSS
qui signifie Cascading Style Sheets (feuilles de style en cascade), est un langage de programmation utilisé en développement web. Il permet de décrire la présentation visuelle des documents HTML

1.Sélecteurs CSS

Les sélecteurs CSS sont utilisés pour cibler les éléments HTML que vous souhaitez styliser
Exemples de sélecteurs :
body {
/* Styles pour le corps de la page */
}

h1 {
/* Styles pour les balises h1 */
}

.maClasse {
/* Styles pour les éléments ayant la classe "maClasse" */
}

#monId {
/* Styles pour l'élément ayant l'id "monId" */
}

2.Propriétés CSS

Les propriétés CSS définissent les styles que vous souhaitez appliquer
Exemples de propriétés :
color: red; /* Couleur du texte */
font-size: 16px; /* Taille de la police */
margin: 10px; /* Marge extérieure */
padding: 5px; /* Remplissage intérieur */

3.Feuilles de style externes :

Il est recommandé de placer vos styles dans un fichier CSS externe
styles.css:
body {
background-color: #f0f0f0;
font-family: Arial, sans-serif;
}

h1 {
color: blue;
}

.maClasse {
border: 1px solid #ccc;
}
index.html:
<!DOCTYPE html>
<html lang="en">
<head>
<link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
<h1 class="maClasse">Titre stylisé</h1>
<p>Paragraphe de contenu.</p>
</body>
</html>
