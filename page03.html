JavaScript
est un langage de programmation qui permet d'ajouter des fonctionnalités interactives aux sites web

1.Incorporation de JavaScript dans HTML :

Vous pouvez inclure du code JavaScript directement dans votre document HTML
Utilisez la balise <script> dans la section <head> ou à la fin de la section <body>
Exemple :
<!DOCTYPE html>
<html>
<head>
<title>Ma Page Web</title>
<script>
// Votre code JavaScript ici br </script>
</head>
<body>
<!-- Contenu HTML -->
</body>
</html>

2.Variables et Types de Données :

JavaScript utilise des variables pour stocker des données
Les types de données courants comprennent les chaînes de caractères, les nombres, les tableaux et les objets
Exemple :
var message = "Bonjour tout le monde !";
var nombre = 42;
var tableau = [1, 2, 3];

3.Les opérateurs :

JavaScript prend en charge divers opérateurs pour effectuer des opérations sur les données,
tels que l'addition (+), la soustraction (-), la multiplication (*), ...

4.Les fonctions

Les fonctions permettent de regrouper des instructions pour les exécuter lorsque nécessaire
Le syntaxe est le suivant :
function nomFonction(parametres) {
instructions;
}

5.Les Événements :

JavaScript réagit aux événements, tels que les clics de souris
Vous pouvez attacher des fonctions à ces événements pour déclencher des actions
Par exemple :
<button onclick="affichResultat()">Afficher</button>
<script>
function affichResultat() {
alert("Resultat");
}
</script>
