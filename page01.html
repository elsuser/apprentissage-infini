HTML
est un langage de contenu utilisé pour structurer et organiser le contenu des pages web

1.La structure du document HTML

<!DOCTYPE html>
<html>
<head>
<title>Le titre souhaité</title>
</head>
<body>
<!--Le contenu-->
</body>
</html>

2.Les balises HTML

<h1> jusqu'a <h6> : Pour les titres
<p> : Paragraphe
<a href="#lien"> : Pour les liens
<img src="la-source-de-l'image" alt="description de l'image">

3.Les attributs

Les balises peuvent avoir des attributs, par exemple :
<a href="#home" target="_blank">Visitez le lien</a>
Ici, la balise <a> a un attribut `href` qui contient le lien en tant que valeur

4.Les listes

Vous pouvez créer une liste ordonnée ou non ordonnée
<ul>
<premier element>
<deuxieme element>
</ul>
--------------------------------
<ol>
<un element>
<un element>
</ol>

5.Les formes

Les formulaires sont utilisés pour recueillir des informations saisies par l'utilisateur, telles que le nom, le prénom...
Le syntaxe est le suivant :
<input type="type">
